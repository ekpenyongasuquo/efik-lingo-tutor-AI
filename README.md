# efik-lingo-tutor-AI
**Imagine Cup 2026 Submission**
**By Ekpenyong Mfon - Ensign Global College**

## Overview
A localized AI system grounded in a 22,939-row dataset to teach STEM subjects in the Efik language using indigenous Vigesimal (Base-20) logic.

## Technical Stack
- **AI Core:** Microsoft Azure OpenAI (GPT-4o)
- **Framework:** Retrieval-Augmented Generation (RAG) principles
- **Data:** 22k+ rows of verified Efik STEM vocabulary
