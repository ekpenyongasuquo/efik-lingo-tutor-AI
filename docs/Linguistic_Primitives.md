# Efik STEM Linguistic Primitives

| English Term | Efik Term | Concept |
| :--- | :--- | :--- |
| Atom | Atọm | The smallest indivisible unit. |
| Cell | Ekpri ubet uwem | The building block of life. |
| Photosynthesis | Ndutịm udia ikọn̄ | Plants making food from sunlight. |
| Gravity | Odudu n̄dụri | The force pulling objects to earth. |
| Liquid | Mmọn̄-mmọn̄ n̄kpọ | Matter that flows like water. |
