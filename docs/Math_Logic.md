# Vigesimal (Base-20) Implementation

The Efik STEM Tutor uses a Base-20 logic engine. 
Standard AI defaults to Base-10, but our grounding layer forces the following:
- 1-20: Standard Efik numerals.
- 40: Aba (2 x 20)
- 60: Ata (3 x 20)
- 80: Anan (4 x 20)
- 100: Ikie (5 x 20)

Logic is taught through "Awak" (Multiplication) and "Bahade" (Division/Sharing).
